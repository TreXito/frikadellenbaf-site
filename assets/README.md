# Site screenshots

These images appear in the **Screenshots** section of the site. To swap one,
replace the file with the same name (PNG, ~16:9, < ~500 KB). If a file is
missing, the site shows a clean "Preview coming soon" placeholder.

| Filename | Shown as |
|---|---|
| `web-panel1.png` | Web Panel — Controls & Dashboard |
| `web-panel2.png` | Web Panel — Auctions, Bazaar & Chat |
| `discord-integration.png` | Discord Integration (slash commands) |
| `legendary-flips.png` | Public Big-Flips Feed |
| `login.png` | Web Panel — Login |
