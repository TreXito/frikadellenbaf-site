# Site screenshots

Drop screenshots here with these exact filenames and they appear in the
**Screenshots** section automatically. Until a file exists, the site shows a
clean "Preview coming soon" placeholder (no broken-image icon).

| Filename | What to capture |
|---|---|
| `web-panel.png` | The web control panel — Controls/Dashboard tab (status, inventory, auctions, chat) |
| `discord-integration.png` | Controlling the bot from Discord — e.g. `/list`, `/auctions`, `/profit`, or the admin panel |
| `login.png` | The web panel login screen |
| `game-view.png` | The live Game View tab (in-game render / inventory) |
| `divine-flip.png` | A divine-tier flip notification (webhook embed) |
| `big-flips-webhook.png` | The public big-flips Discord webhook feed |

**Tips**
- PNG, roughly 16:10 or 16:9, ~1200–1600px wide looks best.
- Crop tightly to the content; the cards add their own border/label.
- Keep file sizes reasonable (< ~500 KB each) so the page stays fast.
